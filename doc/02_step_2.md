# Bước 2: Lập kế hoạch

## 1. Chọn công nghệ
- Frontend: Vue.js hoặc React.
- Backend: Node.js + Express hoặc Laravel.
- Database: MySQL.
- Auth: JWT.
- Lưu trữ video: local server hoặc CDN.

## 2. Phân công công việc nhóm
- UI/UX(Thành): Thiết kế trang chủ, xem phim, dashboard admin.
- Frontend Dev(Nhân): Giao diện + call API.
- Backend Dev(Quý): API phim, user, comments.
- Database Dev(Quý): Thiết kế bảng + tối ưu query.
- Tester(Nhân): Kiểm thử giao diện, API, bảo mật.

## 3. Lập timeline (2 tuần)
- Tuần 1: Thu thập yêu cầu – Thiết kế UI – Thiết kế DB.
- Tuần 2: Code frontend + backend – Test – Deploy thử.

## 4. Phân tích rủi ro
- Video dung lượng lớn → cần tối ưu server.
- Người xem nhiều → cần load balancing.
- Spam bình luận → cần giới hạn tần suất.
